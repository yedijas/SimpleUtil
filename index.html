Intro

This simple utility will help to perform common functions widely used by .NET developers, like converting DataTable to Excel or CSV. I tried to make it as generic as I could to ease the use. For note, this is still using .NET 2.0.

Who Need This

Anyone lazy enough to code those common codes.
